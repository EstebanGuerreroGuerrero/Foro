Instrucciones de Instalacion

1.- Copiar y descomprimir el archivo en la carpeta c:\xampp\htdocs, al final tendras una carpeta llamada “foroprealfa-v01”, a la cual se puede acceder desde el localhost con este nombre
2.- Buscar el archivo schema.sql en foroprealfa-v01/schema.sql, una vez localizado copiar y pegar el contenido en una ventana de MySQL, PHPMyAdmin o MySQL WorkBench
3.- En caso de ser necesario modificar el archivo C:\xampp\htdocs\foroprealfa-v01\admin\core\controller\Database.php para los datos asociados al acceso a la base de datos.
4.- Acceder a http://localhost/foroprealfa-v01/admin/ y entrar al sistema con los siguientes datos.
Usuario: admin
Password: admin

El usuario de la bd por defecto es root.
